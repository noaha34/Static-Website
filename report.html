1. I created this website for my friend who is a graphic designer studying at Towson. I took info from what he sent me as well as his current portfolio website.
2. No images or content other than the images my friend sent me are included in this site. The social media icons adhere to their usage policy of linking only to their site.
3. My site contains 5 pages, the main index page, the about me page, the contact page, the past projects page, and the pricing page. Index is loaded by default.
4. My site includes a header.
5. My site includes navigation links through a dropdown menu, index can be accessed by clicking the name logo.
6. My multicolumn form on the contact page utilizes a media query to shrink to one column on smaller displays like mobile. part of the text fields are cut off though.
7. My site retains coherent style across all pages, I have one parent style sheet that is used for the whole site (including this page) and then individual style sheets for each page.
8. My site includes images that all include alt attributes that accurately describe the image.
9. Each page declares its favicon and is displayed.
10. All my non HTML resources are organized into directories images and css. My javaswcript is minimal and is included at the bottom of my html with script tags.
11. I do not use any third party libraries or frameworks, I reimplemented ideas I saw on W3 schools though.
12. My code contains some errors in Nu that I were unable to completely eliminate.
13. The content is neatly aligned on the page with clear margins on pages. It is left aligned as that is where the menu button is.
14. My site uses two different font faces and styles, arial, sans-serif, and itc-avant-garde-gothic-pro.
15. My site includes a form on the contact page.
16. My site contains links to external sites through the social media embeds at the bottom left corner of the webpage on each page.
17. My source code is readable with accurate spacing that can easily be edited, and follows conventional formatting. My names for my classes and ids represent the element they are effecting.
18. My site includes an animation on the home page (index) when the images are navigated there is a fade in effect that smoothes out the transition between slides. It is also trigerred on opening the website.
19. My site has a git repository that was shared with user twodee.
20. All file names are entirely lowercase and only contain alphanumeric characters.
21. My site has its own domain name: https://project1.noahscorner.tech.
22. My site is served out over HTTPS on port 443 with a certificate from Let's Encrypt.
23. Users that access on port 80 are automatically redirected to port 443.
24. There were a steady flow of commits, however they were mainly during the last few days within the due date.
